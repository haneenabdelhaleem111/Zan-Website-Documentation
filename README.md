# Overview
The Pharmacy Website is a freelance WordPress-based project built for managing products, offers, and customer orders in an online pharmacy. It demonstrates WordPress site architecture, plugin integration, and basic e-commerce functionality using WooCommerce.

Live Demo: https://zan2026.com

## 🖥️ Project Features
# 1. Homepage
Hero section welcoming to the store website
Quick navigation to product categories.
Banners to be added
Best Sellers, New arrivals and the cosmetics sections

# 2. Products Management
Products are added via WordPress admin dashboard.
Categories and tags for easy organization.
WooCommerce integration for pricing, stock management, and checkout.
Product detail pages with images, description, price, and add-to-cart functionality.

# 3. Offers & Promotions
Special offers can be added as separate posts or custom sections.
Displayed on the homepage and product pages.
Configurable expiration dates for promotions.

# 4. Order Handling
Customers can add products to the cart and complete checkout.
Order history accessible in WordPress admin.

# 5. Pages
Home
Shop 
Best Sellers Page
About Us
Contact Us (includes a contact form)
Cart / Checkout (WooCommerce default pages)
